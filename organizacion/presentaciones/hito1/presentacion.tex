\section{Panorama Global}
\begin{frame}
\frametitle{Panorama Global}

Diseñar e integrar un \textbf{sistema de control} que permita el vuelo \emph{autónomo} de un Cuadricópetro comercial.
\pause

\begin{columns}

\column{2.2in}

\begin{figure}[h!]
  \begin{center}
  \includegraphics[width=.85\textwidth]{./Pics/quad.jpg}
  \end{center}
\end{figure}

\column{2.2in}

\begin{itemize}
	\item Generador de rutas\pause
	\item Instrumentación: obtener variables del sistema \pause
	\item Actuador sobre motores
\end{itemize}

\end{columns}
\pause
\vspace{15pt}
No se diseñará ni la mecánica ni la electrónica del sistema.\\
Se integrarán los componentes.\\
Se conservará la posibilidad del manejo manual

\end{frame}

\section{Planificación Original}
% ========================= FRAME ===============================
% ===============================================================
\begin{frame}
\frametitle{Planificación Original}

Al día de la fecha los objetivos planteados son:\pause

\begin{itemize}
\item \textcolor{green}{Elección del \emph{Hardware} necesario para el desarrollo del proyecto: elección del \emph{Quadcopter}, Microprocesador, Instrumentación, Comunicación.} \pause
\item \textcolor{green}{Adelanto del modelado físico.}\pause
\item \textcolor{red}{Caracterización de los motores.}\pause
\item \textcolor{green}{Diseño de los algoritmos de control de los motores.}\pause
\item Programación del microprocesador para comunicarse con PC. \pause
	\begin{itemize}
		\item \textcolor{green}{Comunicación serial}
		\item \textcolor{green}{Comunicación ethernet}
		\item \textcolor{red}{Comunicación Wifi}
	\end{itemize} \pause
\item Programación del microprocesador para comunicarse con la instrumentación.\pause
	\begin{itemize}
		\item \textcolor{green}{Programa en \emph{python} para leer datos en PC}
		\item \textcolor{red}{Portar dicho programa a \textbf{C}}
	\end{itemize}
\end{itemize}

\end{frame}

\section{Dificultades}
% ========================= FRAME ===============================
% ===============================================================
\begin{frame}
\frametitle{Dificultades}

\begin{itemize}
	\item Falta de información acerca de los componentes.\pause
	\item Daño accidental de algunos componentes. \pause
	\item Insuficiencia de la gestión de costos.
\end{itemize}
\pause
El presupuesto para imprevistos no consideraba como probable la avería de ciertos componentes que resultaron dañados.\\
\end{frame}

\section{Replanificación}
% ========================= FRAME ===============================
% ===============================================================
\begin{frame}
\frametitle{Replanificación}

Entre el $1^{er}$ Hito y el 3 de Octubre no hay tareas planificadas.\\ \pause
Dicho tiempo se utilizará para terminar las actividades pendientes.\\ \pause

\vspace{20pt}

\begin{Large}
Plan de contingencia
\end{Large}
\pause
\begin{itemize}
	\item Adquirir repuestos \pause
	\item Reparar componentes dañados
\end{itemize}

\vspace{15pt}
\pause
Dado el avance del proyecto, los objetivos para el segundo Hito son:
\pause
\begin{itemize}
	\item Modelado físico del \emph{Quadcopter} 
	\item Resultados del estudio del vuelo.
	\item Simulador.
	\item Generador de Rutas.
	\item Comenzar el desarrollo de los algoritmos de control.
\end{itemize}
\end{frame}
