\section{Dificultades}
% ========================= FRAME ===============================
% ===============================================================
\begin{frame}
\frametitle{Dificultades}

Se encontraron las siguientes dificultades:
\pause


\begin{itemize}
\item Falta de información acerca de los componentes.
\pause
\item Problemas de comunicación con la electrónica adquirida.
\pause
\item Dificultad para caracterizar ciertos componentes.
\pause
\item Daño accidental de algunos componentes.
\pause
\item Insuficiencia de la gestión de costos.

\end{itemize}
\pause
Debido a estas dificultades no fue posible culminar la caracterización de motores ni la implementación de la comunicación entre el procesador y la instrumentación.\\
\pause
El presupuesto para imprevistos no consideraba como probable la avería de ciertos componentes que resultaron dañados.\\
\pause
Fue necesario obtener más financiación.
\end{frame}

\section{Consideraciones a futuro}
% ========================= FRAME ===============================
% ===============================================================
\begin{frame}
\frametitle{Consideraciones a futuro}

Se plantean las siguientes consideraciones para el resto del proyecto:
\pause


\begin{itemize}
\item Se repondrán los componentes dañados.
\pause
\item Se adquirirán repuestos para componentes para los cuales no se disponía de respaldo o plan  de contingencia inmediato.
\pause
\item Se adquirá de electrónica adicional como plan de contingencia en caso de que no sea posible reutilizar algunos componentes privatidos del cuadricóptero. 
\pause
\item Se replanificarán las actividades.

\end{itemize}
\end{frame}
