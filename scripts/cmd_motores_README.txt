Este script permite enviar el comando a los 4 motores del quad.

Para ello es necesario invocarlo junto con el valor de throttle que se quiere imprimir en los cuatro motores.

Esto es:

sh cmd_motores.sh throttle_0x68 throttle_0x69 throttle_0x6A throttle_0x6B

donde throttle_0xXX es el valor de throttle que se le quiere imprimir al motor XX
